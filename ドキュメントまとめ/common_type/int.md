# Verse の int 型

## 特徴

- 整数を格納し、処理するために使う
- 正数、0、負数を含められる
  - `-9,223,372,036,854,775,808 ～ 9,223,372,036,854,775,807`
  - 小数部分は無い

## int の演算

### 数式演算

- 4 つの基本的な数式演算を利用できる
  - `+`
  - `-`
  - `*`
  - `/`
- `/` のときのみ結果が違う
  - 成功すると `rational` 型が返る
    - `rational` 型は次の関数のパラメータとしてのみ使える
      - `Floor()`
      - `Ceil()`
  - 失敗する可能性がある

## 代替表現

- 16 進数で整数を表すことができる
  - 頭に `0x` をつける
  - 例: `0x7F = 127`

## 実装の詳細

- 現在は **符号付き 64 ビット整数** として実装されている
- 将来的には、意味論的にあらゆるサイズの整数を表すようになる
  - `-2^63, ... ,-1,0,1,...2^63-1`
